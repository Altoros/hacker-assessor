Missing Skills for Senior Front End Developer

  • TDD

    TDD is an art, do, not write a line of code before you have a failing test. This will help you make code that is better for your friends to live with.

    Your current expertice is not enough, your tests should be lighter, faster and more stable, may be it is time to learn "tape".

    Lectures

    • Optimus
    • WAT

  • Vanilla JS

    JavaScript is the most popular programming language in the world. A proper understanding of it's good parts is desirable if you want to write maintainable, concise and usable code that will last forever.

    You don't have to be an expert in vanilla js but your understandig should be wide enough to be able to kick out jQuery.

    Lectures

    • jQuery
    • Codecademy JavaScript Path
